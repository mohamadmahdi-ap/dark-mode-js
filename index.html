dark Light Mode

Light-on-dark color scheme, also called dark mode, is a supplemental mode that uses a color scheme in which the content of a webpage is displayed on a dark background. Such a color scheme reduces the light emitted by screens and enhances readability. Switching to dark mode allows website users to move to an eye-friendly and resource-saving design whenever they want.

Steps to Create Dark/Light mode:

  • Create an HTML Document.
  • Create CSS for the document file as well as for dark mode.
  • Add a switch/toggler to toggle between light and dark modes.
  • Add functionality to the switch/toggler to toggle between light and dark mode using javascript.

style.css
copy
:root {
--background: #ededed;
--title: black;
--text: #222;
--box: #fff;
--box-shadow: #cdcdcd;
}
.dark-theme {
--background: #222;
--title: white;
--text: #e6e6e6;
--box: #2e2e2e;
--box-shadow: #181818;
}
body {
background: var(--background);
}
h1, h2, h3, h4, h5, h6, a {
color: var(--title);
}
p, li {
color: var(--text);
}
script.js
copy
var darkLightBtn = document.getElementById("toggle");
darkLightBtn.addEventListener("click" ,()=>{
document.body.classList.toggle("dark-theme");
})
